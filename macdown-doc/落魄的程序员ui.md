### UI 设计

​         原来网页设计都使用Photoshop（PS） 或 fireworks(网页三剑客之一)。

​         PS有写生疏，网页设计PS也用不了这么多高级的效果，最主要的对H5 （移动）输出比较麻烦（用户人群少）。然后咨询了UI同事

了解到现在用来网页设计都用sketch（草图），然后B站一通瞎找，找了个原型设计，组件分离视频教程，一气呵成，全部看完。
